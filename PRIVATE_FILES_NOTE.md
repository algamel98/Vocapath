# Private Files

The following files contain sensitive personal information and are excluded from this repository:

- `mycheveningapplication.html` - Private Chevening scholarship application page
- `data/chevening_vocabulary.json` - Personal vocabulary database (308 words)
- `fullmodecheveningdata.html` - Full mode study page for Chevening vocabulary

These files are password-protected in the live application and are kept private for security reasons.

## Access

The Chevening application page is accessible through a hidden link in the main website and requires password authentication (password: 99).
